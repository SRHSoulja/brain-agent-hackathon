Brain Agent

Autonomous task orchestration with safety gates. Model-agnostic. File-based. No database, no cloud infrastructure required.

Brain Agent picks up tasks from a file-based queue, decomposes each into steps using an LLM, executes those steps via sandboxed tools, and decides whether to continue or stop. Before every write operation, a verdict gate checks that task state is consistent.

Architecture

Verdict Gate

Every write triggers a state consistency check. Corrupted state blocks the write before damage spreads.

Authority Surfaces

Each task declares which directories it can write to. Writes outside the boundary are rejected.

Escalation Policy

After each task, the agent evaluates whether to continue or stop. Chain depth limits prevent unbounded runs.

Model-Agnostic

Planning and execution work with any LLM. Tested with Claude API and AMD-hosted open models via vLLM.
